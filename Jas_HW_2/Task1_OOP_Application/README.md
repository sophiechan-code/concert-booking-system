# Concert Booking System (Task 1: OOP Application)

## Description
This is a Python-based concert booking system demonstrating core Object-Oriented Programming concepts:
- **Abstraction**: Abstract base class `Person`
- **Inheritance**: `Fan` → `VIPFan`, `Person` → `Artist`
- **Encapsulation**: Private attributes with getters/setters
- **Polymorphism**: `display_info()` overrides
- **Operator Overloading**: `__lt__`, `__eq__` in `Booking` class

## How to Run
1. Ensure Python 3.8+ is installed.
2. Navigate to the `Task1_OOP_Application` folder in terminal.
3. Run:
4. Follow the menu prompts to view concerts, register fans, and book tickets.
